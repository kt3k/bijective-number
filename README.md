# bijective_base_n

Convert numbers to
[bijective base-n](https://en.wikipedia.org/wiki/Bijective_numeration) notation.

```
0 -> (empty string)
1 -> a
2 -> b
...
25 -> y
26 -> z
27 -> aa
28 -> ab
...
52 -> az
53 -> ba
...
701 -> zy
702 -> zz
703 -> aaa
```

This system is used, for example, for labeling columns in Excel.
